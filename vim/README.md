# vim

Based on use [The Ultimate vimrc](https://github.com/amix/vimrc.git)

## 1. Install latest vim

1. download source from github

```sh
git clone https://github.com/vim/vim.git
```

2. build and install

```sh
./configure && make -j
make install
```

2. Install Configurations

```sh
./install_conf.sh
```
